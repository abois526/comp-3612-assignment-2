Description:

This site was developed for an assignment as part of COMP 3612 at Mount Royal University. The main purpose of the assignment is to demonstrate our ability to work with JavaScript before utilizing frameworks in the future. We were assigned the task of developing a single-page application for a clothing website with limited functionality. Key features and functionality have been outlined in the requirements, but the design of the specified pages has largely been left up to our discretion.

Technologies Used:

HTML, Tailwind CSS, JavaScript
